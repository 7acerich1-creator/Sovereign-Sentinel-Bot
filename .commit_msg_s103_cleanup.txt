refactor(S103): delete 1,695 lines of dead local rendering code

rendering code that nothing called anymore. All three pipeline entry
points (/pipeline, /batch, Alfred daily_trend_scan) delegate to the
RunPod pod via withPodSession() + produceVideo(). The local ffmpeg
assembly path was dead weight causing audit confusion.

  music bed, transition stings, captions, brand intro/outro splicing)
- generateSceneImage() -- deprecated local Imagen 4 scene generation
- generateThumbnail() + generateLongFormThumbnail() -- unused local
  thumbnail generators (pod handles thumbnails now)
- generateFallbackGradient() + isValidImage() -- scene image helpers
- uploadAndQueue() -- old Supabase upload (pod does R2 directly)
- SCENE_VISUAL_STYLE, IMAGEN_POSITIVE_DIRECTIVE, IMAGEN_NEGATIVE_BAN,
  BRAND_AESTHETIC_APPEND, brandNegativeBan(), NICHE_FILTERS -- all
  only consumed by the deleted functions
- BRAND_INTRO_DUR, TERMINAL_OVERRIDE_DUR_MIN, computeTerminalOverrideDuration()
  -- old timing constants only used by assembleVideo
- sanitizeFontPathForDrawtext() -- only used by local drawtext rendering
- import { generateCaptionsFromAudio } -- unused import

KEPT: produceFacelessVideo() (pod delegation), generateScript(),
generateStandaloneShorts(), renderAudio(), all types/interfaces,
BrandNicheViolation, produceFacelessBatch().

3,211 lines -> 1,519 lines. Zero tsc errors.
