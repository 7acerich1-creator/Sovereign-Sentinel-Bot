# Voice + Tone — Sovereign Synthesis

**Locked:** 2026-04-26 — after the "vectors" incident proved jargon was filtering out the right audience instead of the wrong one.

---

## The frequency

Sovereign Synthesis copy is written from a sovereign frame: calm, measured, without persuasion theater, without scarcity tactics, without hype, without desperation. The voice does not beg. The voice does not manipulate. The voice describes the architecture clearly and lets the right reader recognize themselves in it.

The reader is treated as an adult who can think. We do not over-explain. We trust them to assemble the meaning. (See `PURPOSE.md` at the repo root for the canonical content rule: *"If what's understood don't have to be explained."*)

---

## The Mom Test — the hard rule

**Before any consumer-facing copy ships, it must pass the Mom Test:** a smart adult who is not in the sovereignty subculture must understand every sentence on first read.

This is a hard rule, not a soft preference. The Mom Test exists because a real subscriber (Christina, 2026-04-26) read "the seven dark psychology vectors mapped in Protocol 77" and had no idea what "vectors" meant. The right word was "tactics" — and once it was changed, the same sentence landed cleanly. Jargon was filtering OUT the right audience instead of filtering IN candidates.

## The three jargon tiers

### Tier 1 — Brand frequency words. KEEP.

These are the words that ARE the product. Cutting them flattens the brand. They earn the reader's attention by being unfamiliar in a specific, deliberate way. The right reader hears them once and snaps awake.

| Word / Phrase | Why it stays |
|---|---|
| Sovereign / Sovereignty | Core brand word |
| Protocol (and Protocol 77) | Product name |
| The Shield | Tier 2 product |
| The Map | Tier 3 product |
| The Architect | Tier 4-6 product line |
| Inner Circle | Tier 7 |
| The formation | Canonical Purpose statement language |
| The simulation | Canonical brand-philosophy term |
| Escape Velocity | Canonical brand term |
| Firmware Update | Canonical brand metaphor |
| The Containment Field | TCF brand name |

### Tier 2 — Specialized but explainable. KEEP, but explain on first use.

These words are concepts the customer learns by reading. They earn their place IF the surrounding sentence makes the meaning land in context. If they don't, replace them.

| Word | Plain-English context required? |
|---|---|
| Architecture | Often clear in context ("the architecture you're running") |
| Lighthouse Stance | Yes — must be explained or in context (lighthouse → ships sort themselves) |
| Gray Rock | Yes — paired with "the four-second pause" or similar |
| The Approval Reflex / The Overload Spiral / The Identity Lock | Yes — these are taught concepts; if used without setup, restate the meaning |
| The 77-day Protocol | Self-explaining once "Protocol 77" is in scope |

### Tier 3 — Pure jargon. CUT EVERYWHERE.

These words add nothing for the reader and actively filter out smart adults who aren't in our subculture. Replace with the plain word that means the same thing. **No exceptions on consumer-facing copy.**

| Cut this | Replace with |
|---|---|
| vector / vectors | tactic / tactics, pattern / patterns |
| transmission vector | tactic, pattern, mechanism |
| heuristic | reflex, pattern, shortcut |
| memetic / memetic warfare / memetic trigger | cultural, message, hook |
| fossil record | leftover, residue, holdover |
| frequency weapon | message, signal, tool |
| codebase | wiring (when it's metaphor), code (when literal) |
| load-bearing | critical, holding things up |
| homeostasis | balance, status quo |
| reciprocity loop | back-and-forth pattern, give-and-take loop |
| anchoring bias | how the first number sticks |
| identity fusion | losing yourself in the role |
| authority hijacking | borrowing other people's authority |
| social proof cascade | the herd effect |
| dark psychology audit | dark psychology check / breakdown |
| dark psychology mechanisms | dark psychology tactics |
| reactive patterns | the things you do without thinking |

If you are tempted to use a Tier-3 word, ask: "Would Mom understand this on first read?" If no, replace it.

## Forbidden across all surfaces

These have always been forbidden. Re-listed here for reinforcement:

- Generic motivation ("You've got this!", "Believe in yourself!")
- Victim-framing without immediate agency follow-up
- Hype without a named mechanism
- Manipulation tactics without inoculation (never use dark psych to extract; only to liberate)
- Competitor bashing
- Desperation energy (urgency yes, desperation never)
- Assumed gender, age, or income bracket of the reader (filter by willingness, not demographics — see PURPOSE.md)

## When this rule conflicts with copy-psychology

The copy-psychology skill describes the structure of persuasion (GLITCH/PIVOT/BRIDGE/ANCHOR). This file describes the WORD CHOICE inside that structure. If a piece of copy is structurally correct (good hook, good pivot) but uses Tier-3 jargon, the copy fails the Mom Test and must be revised.

Structure ≠ vocabulary. Both must pass.
